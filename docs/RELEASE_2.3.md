# Xomacito 2.3 - ¡LA PREMIERE READY UPDATE!!

Xomacito 2.3 corrige la recodificación y mejora la entrega de archivos listos para edición.

## Novedades principales

- La recodificación ya acepta entradas MKV y genera correctamente resultados MP4.
- FFmpeg recibe el contenedor de salida explícito, incluso durante el uso de archivos temporales.
- Las descargas MP4 priorizan audio AAC/M4A para evitar MKV cuando los streams son compatibles.
- El procesamiento individual y por lotes conserva una extensión temporal reconocible.
- El botón **Resultado** abre el Explorador y selecciona directamente el archivo terminado.
- Al finalizar una descarga, Xomacito abre automáticamente la ubicación del resultado.
- Se eliminó la opción redundante **Enviar a Estudio de Imagen** del menú de resultados.
